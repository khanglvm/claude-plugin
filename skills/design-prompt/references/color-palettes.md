# Color Tone Palette Gallery

16 named tone families with emotional descriptions and 3 intensity variants each. Use in Phase 1 (tone selection) and Phase 3 (CSS variable expansion). Each intensity: **whisper** = light/airy, **medium** = balanced default, **bold** = saturated or dark.
